Está precisando de uma logo, um site ou até mesmo um sistema para o seu negócio?

O que é o Portal de Demandas?

O Portal de Demandas foi feito para ONGs, microempreendedores e empresas, enviarem suas demandas para o nosso grupo de estudantes, onde iremos desenvolver o seu site ou até mesmo criar um sistema para o seu negócio.

Totalmente gratuito, em parceria com o Instituto Federal Bragança Paulista.
